## Option 1: Wall-Mounted Tablet + Free App (Best Value)

This is the most popular "Skylight alternative without subscription" for a reason: same wall presence, zero ongoing cost, more functionality.

### What You Need

| Component | Example | Cost |
|-----------|---------|------|
| Tablet | Amazon Fire HD 10 (refurbished) | $80–120 |
| Wall mount | WALI tablet mount or 3M strips | $15–25 |
| Free family app | Honeydew, Google Calendar, TimeTree | $0 |
| **Total** | | **$95–145 one-time** |

### Best Free Apps for This Setup

| App | Best For | Key Features | Limitations |
|-----|----------|--------------|-------------|
| **Honeydew (free tier)** | Families wanting AI + automation | AI list generation, voice input, multi-family groups | Some AI features require paid tier |
| **Google Calendar** | All-Google households | Native integration, widgets | No lists, no family-specific features |
| **TimeTree** | Simple shared calendar | Clean UI, cross-platform | No lists, no AI |
| **Cozi (free)** | Basic calendar + lists | Familiar interface | Ads, no AI, no voice |

### How to Set It Up (15 Minutes)

1. **Get a tablet.** Amazon Fire HD 10 ($80–120 refurbished) is the best value. iPad Mini works if you already have one.
2. **Install your family app.** Download Honeydew, Google Calendar, or TimeTree.
3. **Enable "kiosk mode."** On Fire tablets, use "Show Mode" to keep the screen on. On iPads, use Guided Access.
4. **Mount it.** WALI tablet mounts ($15) work well. Or use 3M Command strips for renters.
5. **Plug in power.** Route a USB cable behind the mount.

**Pro tip:** Set the tablet to auto-brightness and enable "dark mode at night" to avoid a glowing screen at 2am.

## Option 2: Raspberry Pi DIY Calendar (Most Customizable)

For tech-savvy families, a Raspberry Pi + old monitor creates a fully custom family dashboard with zero subscription.

### What You Need

| Component | Cost |
|-----------|------|
| Raspberry Pi 4 | $55 |
| MicroSD card (32GB) | $10 |
| Old monitor or TV | $0 (repurpose) or $50–100 |
| VESA mount | $15–20 |
| **Total** | **$80–185 one-time** |

### How It Works

1. Install **DAKboard** (free tier) or **MagicMirror** (open source)
2. Connect Google Calendar or iCal feeds
3. Display on any screen
4. Customize with weather, photos, to-dos

**Limitation:** Requires technical setup (1–2 hours). No AI planning or family-specific coordination.

## FAQ

### Can I get Skylight features without paying the subscription?
Yes. A wall-mounted tablet with a free app like Honeydew provides calendar display, lists, and more features than Skylight—without any subscription.

### What's the cheapest Skylight alternative?
A refurbished Amazon Fire HD 10 ($80) + wall mount ($15) + free Honeydew app = $95 total, no subscription. That's $440+ less than Skylight over 5 years.

### Do any wall calendars have no subscription at all?
Echo Show 15 ($250) and Google Nest Hub Max ($230) display calendars without subscriptions. But they're smart displays first, calendars second.

Only if you value the premium hardware aesthetics and want zero setup. For most families, a tablet + app provides more value at lower cost.

### Can I use an old iPad as a Skylight alternative?
Yes! An older iPad (even iPad 2) works fine for calendar display. Use Guided Access to keep it on one app, and mount it on the wall.
